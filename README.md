# crime-life-3-WidescreenFix  
A WidescreenFix for the game [Crime Life 3](https://gamejolt.com/games/crimelife-3/4274)  
Features: Widescreen Fix (Ultrawide Support), MSAA, Dual Pass, Texture Filter, MipMaps, Anisotropic filter, Models Rendering Fix, Increased NPC and Vehicles Spawn.  
Requires the d3d8 asi loader from [here](https://github.com/ThirteenAG/Ultimate-ASI-Loader/releases)
